Logo embarqué : il est intégré directement dans le code (base64), donc aucune URL à gérer : il s'affiche tel quel dans la plupart des messageries (Apple Mail, Outlook, Thunderbird…).
• Dans Gmail web, l'éditeur retire parfois une image collée. Si le logo disparaît, héberge le fichier Logo Unkai noir.png sur unkaiweb.fr et remplace le src de l'image par son adresse.
• Vérifie l'e-mail affiché (contact@unkaiweb.fr) et le titre (Créateur de sites web sur mesure, modifiable).
Pour l'installer : ouvre cette page, sélectionne la signature (entre les deux traits), copie (Ctrl/Cmd+C) et colle dans Gmail (Paramètres → Signature) ou Outlook.
